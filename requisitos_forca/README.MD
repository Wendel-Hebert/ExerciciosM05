# Jogo da Forca – Documentação de Requisitos

## Propósito da Atividade

O objetivo desta atividade é realizar o levantamento e a documentação dos requisitos para o desenvolvimento de um **Jogo da Forca Online**.  

O sistema permitirá que usuários joguem o tradicional jogo da forca diretamente pelo navegador, oferecendo tanto **modo singleplayer** quanto **modo multiplayer online**.

Durante a atividade foram definidos:

- **Requisitos Funcionais (RF)** – funcionalidades que o sistema deve possuir.
- **Requisitos Não Funcionais (RNF)** – critérios de qualidade, desempenho e segurança.
- **Restrições do sistema** – limitações técnicas ou arquiteturais do projeto.

Essa documentação servirá como base para as próximas etapas do desenvolvimento do sistema.
